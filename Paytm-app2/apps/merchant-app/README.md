# Merchant App

`apps/merchant-app` is the merchant-facing frontend for `Paytm-app2`. It is meant to represent the merchant side of the payment system, separate from the end-user wallet experience handled by `apps/user-app`.

This app currently uses Next.js, NextAuth, and the shared packages from the monorepo. It is expected to evolve into the merchant dashboard and merchant interaction surface as the broader payments simulation becomes more complete.

For the full monorepo overview, architecture, and roadmap, see the root [README](../../README.md).
