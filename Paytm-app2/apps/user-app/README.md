# User App

`apps/user-app` is the end-user frontend for `Paytm-app2`. It is responsible for the wallet-side experience: authentication, checking balances, adding money, viewing transaction activity, and peer-to-peer transfer flows.

This app currently uses Next.js, NextAuth, shared UI/state packages from the monorepo, and Prisma-backed data from `packages/db`.

For the full project overview, architecture, roadmap, and workspace details, see the root [README](../../README.md).
